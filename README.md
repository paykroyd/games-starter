# Javascript Game Example

## Instructions

Start the web server:

```
./start-server.sh
```

In your browser go to: [http://localhost:8000](http://localhost:8000).

## How to make a game

This is using [Crafty.js](https://craftyjs.com) as a game engine.

## Games Available

- [Pong](http://localhost:8000/pong.html)
